# Simple ToDo App

## Description
This ToDo app allows you to add, remove, and edit tasks. It's built using HTML, CSS, and JavaScript.

## How to Run
1. Download the project files.
2. Open the `index.html` file in a web browser.

## Features
- Add new tasks.
- Remove tasks.
- Edit existing tasks.

## Technologies Used
- HTML
- CSS
- JavaScript
